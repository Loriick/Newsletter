# Newz Later 😁

## What I did

A newletter page that is allow the user to register or subscribe to a newletter

## What I used

- React
- Nextjs
- Chakra-ui
- Styled-component

## How to use it

### Clone this repo

### Install dependencies

```
pnpm install
```

### Start the project

```
pnpm run dev
```

Alternative use the command line `ni` to install or `nr` to start the project
